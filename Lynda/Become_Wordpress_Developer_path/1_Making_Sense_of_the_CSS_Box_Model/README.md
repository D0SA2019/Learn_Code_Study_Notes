# 1. Making Sense of the CSS Box Model
***by Morten Rand-Hendriksen***

Based on old-school typography blocks, the CSS box model is the key to modern web design. Once you master it, you can build any layout you wantâ€”and troubleshoot most of the common CSS alignment and positioning woes. In this course, Morten Rand-Hendriksen offers a "four course meal" approach to the CSS box model, breaking it down into easily digestible chunks.

He covers techniques such as styling text and images, floating content, using display properties, and positioning elements. He'll also show you how to take your website beyond boxes with advanced CSS tools and techniques. Each lesson is supported by real-world demonstrations of the concepts in action, and Morten closes with a "nightcap" that will get you thinking about going further with CSS.
